Was kann ich tun?

Fragen Sie Ihre:n Abgeordnete:n, wie er oder sie zu automatischer Gesichtserkennung steht. Fragen Sie außerdem, ob er oder sie für ein Verbot von Gesichtserkennung im Bundestag stimmen wird.


Dazu können Sie Bundestagsabgeordnete auf Social Media oder per E-Mail kontaktieren.


Alternativ können Sie abgeordnetenwatch.de nutzen. Die Fragen und Antworten sind transparent für alle sichtbar. Damit geben Abgeordnete eine Zusage ab, die sie später kaum mehr zurücknehmen wollen.
